# BERS 2026 Workshop Page

This directory contains a static workshop homepage for **BERS 2026**.

## Files

- `index.html`: main one-page workshop site
- `styles.css`: site styling
- `BERS2026.docx`: source content provided for the page

## Deploy to GitHub Pages

1. Create a GitHub repository and upload these files to the repository root.
2. In GitHub, open `Settings -> Pages`.
3. Set `Build and deployment` to `Deploy from a branch`.
4. Choose the branch containing these files and set the folder to `/ (root)`.
5. Save the settings and wait for GitHub Pages to publish the site.

## Local Preview

Open `index.html` directly in a browser, or run a local static server from this folder.
